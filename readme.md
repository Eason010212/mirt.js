**mirt.js** is a high-performance JavaScript library designed for Multidimensional Item Response Theory (MIRT) modeling directly in the browser or Node.js. It enables researchers and developers to estimate person parameters (latent traits) and item parameters for complex, multi-construct assessments.

### Key Features

* **Multidimensional Support:** Handle Compensatory and Non-compensatory models.
* **Parameter Estimation:** Support for Expectation-Maximization (EM) and Stochastic Gradient Descent (SGD) via TensorFlow.js.
* **Scoring Engines:** Implement EAP (Expected A Posteriori) and MAP (Maximum A Posteriori) scoring.
* **Zero Latency:** Perform complex psychometric scoring on the client side without sending raw data to a server.
